## Introduction

SuperHero Hunt is a project that aims to provide superhero enthusiasts with an easy-to-use platform to explore their favorite characters. The application is built using HTML, CSS, JavaScript, and leverages the Marvel API to fetch superhero data and images. Users can search for superheroes, view their profiles, and even add them to their favorites list.

## Features

- Browse and search for superheroes
- View detailed profiles of superheroes, including images and descriptions
- Add superheroes to your favorites list
- Remove superheroes from your favorites list
- Responsive design for optimal viewing on various devices

## Getting Started

To get started with SuperHero Hunt, follow these steps:

1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Start exploring superheroes!

## Usage

- Click on the "SEARCH" button in the navigation bar to search for a specific superhero.
- Explore the list of superheroes on the homepage and click on a superhero card to view more details.
- Click on the "Add to Favorite" button to add a superhero to your favorites list.
- Use the search input to find superheroes based on their names or keywords.
- Navigate to the "Favorites" page to view your saved superheroes.
